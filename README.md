# Agentic Growth Lab

ROI-obsessed digital marketing partner built with Next.js 14. This single-page experience showcases services across Meta Ads, Google Ads, SEO, lifecycle automation, and revenue intelligence with a focus on measurable business outcomes.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the site.

## Production

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- CSS Modules via global stylesheet

## Deployment

Ready for deployment on Vercel. Use the provided production command to ship: `vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-70146dc4`.
